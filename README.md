- bọn cờ hó fpt đổi địa chỉ ip public

## Bắt đầu

Đầu tiên chạy npm install/yarn add

```bash
npm i

yarn add
```

Sau đó chạy lệnh run dev để khởi động môi trường dev

```bash
npm run dev

yarn dev
```

Mở [http://localhost](http://localhost) trong trình duyệt web của bạn
